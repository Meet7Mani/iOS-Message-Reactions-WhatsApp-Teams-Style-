# iOS-Message-Reactions-WhatsApp-Teams-Style-
A reusable reaction manager for iOS chat applications that allows users to react to messages with emojis—similar to WhatsApp and Microsoft Teams.
---

## 🚀 Features

* Add reactions (👍 ❤️ 😂 etc.) to messages
* Toggle reactions (add/remove)
* Support multiple reactions per message
* Clean and reusable architecture
* Easy integration into chat modules

---

## 🧠 Concept

Each message can hold a list of reactions.

A reaction consists of:

* emoji
* user identifier
* timestamp (optional)

The manager handles:

* adding new reactions
* removing existing ones
* updating UI state

---

## 📱 Use Case

Perfect for:

* chat applications
* social feeds
* collaborative apps

---

## 🧩 Implementation

The `ReactionManager` handles all reaction logic:

* Add reaction to message
* Remove reaction
* Check if user already reacted
* Update reaction counts

---

## 💡 Example Flow

1. User long-presses a message
2. Reaction options appear
3. User selects emoji
4. Reaction is added or toggled
